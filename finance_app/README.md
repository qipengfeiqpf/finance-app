## 功能

### 收支管理

- 收入记录
- 支出记录
- 分类管理
- 日期记录
- 备注

### 数据分析

- 收支柱状图
- 分类占比饼图
- 时间筛选
- 收支统计

### 预算

- 月预算
- 分类预算
- 超支提醒

### 数据管理

- 编辑记录
- 删除记录
- 搜索
- CSV 导出
- JSON 导入
- JSON 导出
